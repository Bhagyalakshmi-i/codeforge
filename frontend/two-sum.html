Two Sum

Given an array of integers nums and a target, return indices of the two numbers such that they add up to target. Each input has exactly one solution.

⏱ Time O(n)
💾 Space O(n)

def two_sum(nums, target):
    seen = {}

    for i, n in enumerate(nums):
        diff = target - n
        if diff in seen:
            return [seen[diff], i]
        seen[n] = i

// KEY INSIGHT

Use a hash map to store each number and its index. For every element, check whether the complement (target - current) already exists. Single pass → O(n) time.

// FULL APPROACH

Store each number in a hash map as you scan the array. For every element, compute its complement and check whether it has already appeared. Once you find the complement, return the stored index and the current index.

def two_sum(nums, target):
    sorted_nums = sorted(enumerate(nums), key=lambda x: x[1])
    l, r = 0, len(sorted_nums) - 1

    while l < r:
        s = sorted_nums[l][1] + sorted_nums[r][1]
        if s == target:
            return [sorted_nums[l][0], sorted_nums[r][0]]
        elif s < target:
            l += 1
        else:
            r -= 1

def two_sum(nums, target):
    for i in range(len(nums)):
        for j in range(i + 1, len(nums)):
            if nums[i] + nums[j] == target:
                return [i, j]
