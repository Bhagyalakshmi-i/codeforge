Climbing Stairs

Climbing a staircase with n steps. You can climb 1 or 2 steps at a time. How many distinct ways can you reach the top?

⏱ Time O(n)
💾 Space O(1)

FEATURED Fibonacci Bottom-Up O(1) space 
def climb_stairs(n):
    if n <= 2:
        return n

    a, b = 1, 2
    for _ in range(3, n + 1):
        a, b = b, a + b

    return b

// KEY INSIGHT

ways(n) = ways(n-1) + ways(n-2), exactly Fibonacci. Use two rolling variables for O(n) time and O(1) space.

// FULL APPROACH

Every step can be reached either from one step below or two steps below. So the total ways to reach step n is the sum of ways(n-1) and ways(n-2). This makes it a Fibonacci-style recurrence.
