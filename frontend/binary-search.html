EASY Binary Search

Binary Search

Given a sorted array and target, return the target's index. Return −1 if not found. Must run in O(log n) time.

⏱ Time O(log n)
💾 Space O(1)
👁 3,100
💡 12

def binary_search(nums, target):
    l, r = 0, len(nums) - 1

    while l <= r:
        mid = l + (r - l) // 2

        if nums[mid] == target:
            return mid
        elif nums[mid] < target:
            l = mid + 1
        else:
            r = mid - 1

    return -1

// FULL APPROACH

Maintain left and right pointers. Check midpoint each step. If target > mid → search right half; else search left half. Halves search space each iteration → O(log n).

def binary_search(nums, target, l=0, r=None):
      if r is None:
          r = len(nums) - 1
  
      if l > r:
          return -1
  
      mid = l + (r - l) // 2
  
      if nums[mid] == target:
          return mid
      if nums[mid] < target:
          return binary_search(nums, target, mid + 1, r)
      return binary_search(nums, target, l, mid - 1)
